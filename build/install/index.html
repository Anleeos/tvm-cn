安装 TVM¶

查看 install TVM from source。 从源码安装 TVM 使得从官方源码版本有效配置构建变得很灵活。如果您对部署到移动端或嵌入式设备感兴趣， 则无需在设备上安装整个 TVM 堆栈，只需利用 runtime，并通过 deployment and integration guide 安装即可。

想要快速运行 TVM 或访问其他 Demo 和教程，请查看 install from Docker。或者从第三方 TLCPack Python and Conda packages 中预先构建。
